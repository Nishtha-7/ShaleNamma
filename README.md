📚 Shale-Namma Pride

An AI-powered educational awareness platform designed to bridge the information gap between government schools and rural parents by showcasing real-time school improvements, facilities, and achievements.

🚀 Project Overview

Many government schools have significantly improved infrastructure such as smart classrooms, labs, and digital learning tools. However, parents in rural areas are often unaware of these developments, leading them to prefer expensive private schools.

Shale-Namma Pride solves this problem by providing a digital platform that highlights the true progress of government schools using modern technologies like GenAI, multimedia content, and interactive updates.

🎯 Problem Statement

Parents in rural areas lack visibility into the current status and improvements of government schools. This leads to:

Misconception that government schools are low quality
Increased financial burden due to private school admissions
Underutilization of improved public education facilities
💡 Solution

This application provides:

📸 Real-time updates of school facilities (photos/videos)
🏫 Detailed school profiles
🤖 AI-based recommendations and summaries (GenAI integration)
📢 Notifications about school improvements and events
🌐 Easy access for parents and students in rural areas
✨ Key Features
📱 Simple and user-friendly Android interface
🏫 Government school profile dashboard
📊 Facility and infrastructure showcase
🤖 GenAI-powered insights and summaries
🔔 Push notifications for updates and announcements
🌍 Designed for rural accessibility (low complexity UI)
🛠️ Tech Stack
Frontend: Android (Java / Kotlin)
Backend: Firebase / Cloud services (if used)
AI Integration: Generative AI APIs
Database: Firebase Firestore / Realtime Database
Storage: Firebase Storage
Authentication: Firebase Auth (optional)
